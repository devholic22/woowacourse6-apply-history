## 기능 명세 정의
### 횟수
- [ ] 조건: 양수여야 한다.
### 자동차
- [ ] 이름을 가진다.
- [ ] 위치를 가진다.
- [ ] 무작위 값이 4 이상일 시 위치 증가를 요청한다.
### 이름
- [ ] 쉼표를 기준으로 분할된다.
- [ ] 조건: 5자 이하이다.
### 위치
- [ ] 무작위 값이 4 이상일 시 위치가 증가된다.
### 무작위 값
- [ ] 0 ~ 9 사이의 값을 제공한다.
### 입력
- [ ] 이름 입력
- [ ] 횟수 입력
### 출력
- [ ] 자동차 이름 입력 요청
- [ ] 플레이 횟수 입력 요청
- [ ] 실행 결과 출력
- [ ] 각 차들의 상태 출력
- [ ] 최종 우승자 출력
